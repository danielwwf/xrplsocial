A lightweight public page that lists upcoming X (Twitter) Spaces relevant to the XRPL community.

## What it does

- Shows a clean, dark-mode list of upcoming Spaces (title, time, host)
- Each entry links directly to the Space on x.com
- Anyone can get a Space listed by mentioning **[@HerbertXRPL](https://x.com/herbertxrpl)** on a Space announcement tweet
- The page is refreshed automatically (updates are pushed every ~5 minutes)

## Notes

- Times are displayed in **UTC**
- This repository only contains the static site output (HTML + assets)
